## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) version 20.0 or above

### Installation

```bash
npm install
```

### Local Development

Start the development server:

```bash
npm run start
```

This command starts a local development server at `http://localhost:3000/` and automatically reloads when you make changes.

### Build

Generate static content for production:

```bash
npm run build
```

This command generates optimized static HTML files into the `build` directory, ready for deployment.

### Serve Locally

Preview the production build locally:

```bash
npm run serve
```
